# sharemovie
シェアーハートムービー

## サイト概要
### サイトテーマ
・映画を検索でき、その評価を共有できる。

### テーマを選んだ理由
・映画が好きで見た映画の評価をどこかにのっけたいと思ったため。その評価をサイト利用者で共有したいと思ったため。
・自分の見たい映画が簡単に探せたらいいなと思った。

### ターゲットユーザ
・映画が好きな人
・映画を調べたい人

### 主な利用シーン
・見たい映画の検索
・見た映画の評価
