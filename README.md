# sharemovie

## サイト概要
### サイトテーマ
・映画を検索でき、その評価を共有できる。

### テーマを選んだ理由
・映画が好きでその評価をサイト利用者で共有したいと思ったため。

### ターゲットユーザ
・映画が好きな人
・映画を調べたい人

### 主な利用シーン
・見たい映画の検索
・見た映画の評価
